# JISP Explainability

> Status: placeholder — to be populated in later steps.

JISP is **explainable by design**. Every model output is accompanied by:

1. **Feature-level attribution** (SHAP) — which inputs drove the result.
2. **Narrative reasoning** (`reasoning/` via Ollama + Llama 3.3) — a plain-language
   explanation aimed at pre-field decision makers.

Explanations are not optional. They are first-class artifacts of JISP.
